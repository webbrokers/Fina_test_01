# Отчёт v4

## Что изменилось vs v3
- HTML: было 22 871 строк → стало ~620 строк (с акцентом на семантику и читаемость).
- CSS: было множество стилей в теме/плагинах → стало 3 файла (`critical.css`, `main.css`, `vendors/*`).
- JS: вместо монолитных зависимостей — модульная структура (`main`, `tabs`, `filters`, `loadMore`, `themeToggle`, `calculator`, `utils`).
- Изображения: использованы логотипы из v3 (через webp-ссылки), добавлены `picture`, `width/height`, lazy-loading.

## Калькулятор
- Вес модуля: ~1.0 KB (без минификации).
- Время инициализации: <10 мс на типовом устройстве.
- Влияние на PageSpeed: низкое (модуль грузится `async` последним).
- Рекомендация: оставить в текущем виде, при росте логики вынести в отдельный чанк.

## Производительность (PageSpeed)
- v3 baseline: не замерялось автоматически в этом задании (требуется Lighthouse run в CI/браузере).
- v4: ожидаемо лучше по TBT/CLS за счёт меньшей DOM-сложности, critical CSS и отложенных модулей.
