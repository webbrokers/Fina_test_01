## ✅ Чек-лист технического SEO

- [ ] Title оптимален (50-60 символов)
- [ ] Meta description есть и информативен
- [ ] H1 один, релевантен
- [ ] Canonical корректен
- [ ] Alt-тексты у всех значимых картинок
- [ ] Structured data присутствует
- [ ] Мобильная версия корректна
- [ ] Страница индексируема (robots.txt, meta robots)

## ✅ Чек-лист производительности

- [ ] Нет render-blocking JS в `<head>`
- [ ] CSS не блокирует рендер
- [ ] LCP-изображение preload
- [ ] Изображения в WebP
- [ ] Шрифты с `font-display: swap`
- [ ] Critical CSS inline
- [ ] Сторонние скрипты отложены (аналитика, реклама)
- [ ] Нет неиспользуемых библиотек
